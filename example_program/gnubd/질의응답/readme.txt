다용도 체크 서치 플러그인 [20203020 장석환]

※준비사항
-게시판을 2개를 생성(정보가 제공되는 노출형 게시판A, 문의를 남기는 게시판B);
-게시판A는 (_quize) 설정
-게시판B는 (_question) 설정

※적용 및 사용 방법
-게시판A 에 '연동게시판'에 게시판B등록
-게시판A 에 '질문등록'에 질문명,옵션,선택방법를 입력
+게시판A 에  계산식 등록 
-원하는 부분에 latest('스킨은:"quiz_latest"')로 출력
01.latest로 출력한부분은 질문 답변하기, 
02.해당사항에 맞는 게시판A의 게시글들 출력
03.문의하기에 해당내용을 입력하면 게시판B에 등록 

+지원한도계산클릭시 계산식이 추가로 등록


※응용활용

-문의를 없애고 결과 값을 바로 노출 시켜서 문제나 질문을 맞추면 바로 결과가 보여주는 기능변경가능
-문의를 해당 게시글의 커멘드로 update해줘서 관리할수도 있음
