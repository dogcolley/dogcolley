명령어 모드, 입력모드, 마지막행 모드

명령어 모드(command mode)
i: 커서부분으로부터 입력모드
a: 커서부분으로 다음위치에 입력모드
o: 커서다음줄에 입력모드
x: 커서우치 글자 삭제
dw: 커서위치 
dd: 커서 한줄삭제
u: 명령 취소
yy 한줄 복수
p: 붙어놓기
k: 위
j: 아래
l: 오른쪽
h: 왼쪽
o: 맨합줄
$: 맨뒷줄
(): 문장 맨 앞뒤
{}: 문단 맨 앞뒤
숫자 -, 숫자 + : 숫자만큼 윗줄 , 아랫줄 이동
r : 한문장변경
cc : 커서에 있는 밑줄 내용 변경

입력모드(insert mode)이건 자유롭게 쓰는거니깐 esc로 탈출하면 다시 명령모드로 돌아옴

마지막 행 모드(last line mode) 마지막으로 어떻게 해주는걸 정하는 명령어들

w : 파일저장
w '파일이름' : 해당파일 이름으로 저장해준다.
q : 종료
q! : 강제종료 '!'이거 자체가 강제종료뜻임
wq : 저장후 종료
f : 파일변경
숫자 : 해당라인으로 커서이동
$ : 맨끝줄로이동
e! : 맨 윗줄로이동
/문자열 : 현재커서위치에서 부터 앞쪽파일 탐색
?문자열 : 현재커서위치에서 뒤쪽의 문자열 탐색
set nu : vi 라인번호출력
set nonu : vi 라인번호 출력취소
