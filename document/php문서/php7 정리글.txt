PHP7에 추가된점
: < = > 자연비교 연산잔
: 추상 구문 트리 : – 코어의 또 다른 큰 변화. 그러나 개발자에게는 가장 많이 보입니다. PHP 코드를 더 쉽게 파싱하고, 오류를 줄이며, 미래에 더 나은 것을 구축하는 방법을 제공합니다.
: Closure :: call () : 이것은 코어에 추가 된 새로운 메소드입니다. 사전 바인딩보다는 호출 시간에 변수 바인딩을 허용합니다.
: Null Coalesce 연산자 : – 병합 연산자 또는 ?? 첫 번째 피연산자가 존재하거나 널인 경우이를 반환하는 코어에 추가됩니다.


PHP7에  제거점
: mysql_ * 함수는 기본적으로 확장이 제거 될 때 완전히 제거됩니다.
: ASP 태그 (< %) 및 스크립트 태그 (< script language = "php"> < / script >)의 지원이 제거되었으므로 PHP 태그를 입력 할 때 해당 태그를 사용할 수 없습니다.

Http 반환 
200 OK 요청이 정상적으로 완료되었음을 나타낸다.
302 Found 요청된 리소스가 일시적으로 다른 URI에 속해 있음을 나타낸다. 리다이렉트에 이용된다. 401 Unauthorized 사용자 인증에 실패했음을 나타낸다. 
403 Forbidden 접속 권한이 없기때문에 서버가 요청 실행을 거부했음을 나타낸다. 
404 Not Found 요청 URI와 일치하는 리소스를 찾지 못했음을 나타낸다. 
500 Internal Server Error 서버측 프로그램 실행과정에서 에러가 발생했음을 나타낸다. 
503 Service Unavailable 과도한 접속으로 인한 일시적 서비스 불가현상을 나타낸다.
